PiFrame
============
PiFrame is a picture frame service and webapp for the Raspberry Pi.  

dependencies
------------
  npm
  bower
  brew

depends on 
  imagemagick (brew install imagemagick)

build 
-----

```bash
  npm update
  bower update
```

Configure & run
———————————————

```bash
  chmod 777 pics
  chmod 777 slides
  chmod 777 piserver/public/thumbnails
```
